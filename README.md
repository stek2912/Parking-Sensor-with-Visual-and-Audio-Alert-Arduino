Parking Sensor Project (Arduino)

All the Necessary Components:

1. Arduino Uno
2. Ultrasonic Sensor (HC-SR04).
3. BreadBoard
4. 1 kohm Resistor.
5. Green LED.
6. Red LED.
7. Piezo Buzzer
